Brained Crud application


MERN Stack is used in this application to create, read, update and delete data input. 

Other Requirnment- Bootstrap

* To Start Application 

1) cd to brained_client

cd brained_client

2) To start it on server run

npm start


* To Start Database

1) cd to brained_server

cd brained_server

2) To start server run

nodemon app.js



Database is created on mongodb atlas.
